# Personal Finance Manager

A simple command-line app built in Python to track daily expenses, save them to CSV, view reports, and backup/restore data.

## Features
- Add expenses with amount, category, date, description
- View all expenses
- Generate total, average, and category-wise reports
- Data saved automatically to expenses.csv
- Backup and restore using backup.csv

## How to Run
1. Install Python 3.8+
2. Open folder in terminal
3. Run: `python main.py`
